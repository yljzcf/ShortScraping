ShortScraping 设置中心

这里用于维护网页订阅、定时任务、翻译接口和数据存档配置。配置文件按职责拆分，避免互相混用。

📄 配置文件

网页订阅、定时任务和翻译接口分别来自项目 config/ 目录下的独立 JSON 文件。扩展启动、刷新扩展或点击“重新读取配置”时，会从这些文件恢复配置。

  • 网页订阅：config/tag.json
  • 定时任务：config/cron.json
  • 翻译接口：config/trans.json
📌 当前配置概览
🌐 网页订阅

订阅规则来自规则目录 config/tag.example.json，标签随规则固定；勾选需要的订阅并保存，结果写入 config/tag.json。暂不支持自由添加 URL，新增规则请编辑目录文件后重载扩展。

规则目录为空或加载失败，请检查 config/tag.example.json。

说明：浏览器扩展不能直接写项目文件。保存时会先更新扩展本地配置，并尝试通过本地同步服务写回 config/tag.json；若同步服务未启动，会提示仅本地临时生效。

⏰ 定时任务

对应 config/cron.json。Cron 为 5 段格式：分钟 小时 日期 月份 星期，例如 45 * * * *。

🌍 翻译接口

对应 config/trans.json。这里只维护翻译接口、模型、提示词、批量大小、延迟和超时；定时任务请在 config/cron.json 中配置。

🔌 API 模式

免费 API 或兼容翻译接口。AI 模式不会使用这里的地址。

🤖 AI 模式

AI 接口按 OpenAI 兼容 Chat Completions 格式调用，请妥善保管 API Key，避免提交真实密钥。
